# devsec-cli
A command line tool for DevSecOpso

## Usage
```
Usage: devsec gitlab [OPTIONS] --auth <GITLAB_TOKEN> --output <FILE>

Options:
      --auth <GITLAB_TOKEN>
      --org <Group id>
      --projects             Get all projects in a group
  -o, --output <FILE>
  -h, --help                 Print help
```
